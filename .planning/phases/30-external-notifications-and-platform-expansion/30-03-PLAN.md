# 30-03 Plan: Bounded Channels And Spaces Design

## Goal

Define platform expansion without turning Chatify into a broad clone product.

## Tasks

- [x] Scope private invite-only spaces as the first possible expansion.
- [x] Preserve username privacy, membership authorization, moderation, and notification boundaries.
- [x] Reject public directories, anonymous posting, and federation for the first platform phase.

## Acceptance

- Channels/spaces are bounded and compatible with the existing private messenger model.
