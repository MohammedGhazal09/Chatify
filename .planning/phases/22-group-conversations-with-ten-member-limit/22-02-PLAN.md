# Plan 22-02: Frontend Group Creation UI

## Objective

Let users create groups from the existing new-chat entry point using group name and username chips.

## Tasks

1. Add `CreateGroupChatPayload` and `chatApi.createGroupChat`.
2. Add `useCreateGroupChat` mutation that merges returned group chats into the existing chat cache.
3. Extend `NewChatDialog` with Direct and Group modes.
4. Add group name input, username chip input, remove buttons, member counter, and create-group submit button.
5. Add group submit handling in `chat.tsx`, preserving unrelated local chat work.
6. Keep calls disabled for group chats with honest copy if not already covered.
