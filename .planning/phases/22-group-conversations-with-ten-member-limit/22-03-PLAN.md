# Plan 22-03: Group Realtime And Cache Convergence

## Objective

Prove existing message, socket, unread, attachment, and cache paths work for groups without direct-chat assumptions.

## Tasks

1. Audit socket room join, typing, message, delivery, read, edit, delete, reaction, call rejection, and `chat:new` behavior for group chats.
2. Add focused backend/socket tests for group membership authorization and realtime delivery.
3. Add message tests proving group members can send/read/search and outsiders cannot access group messages.
4. Verify frontend cache receives group `chat:new` and message socket events without direct-chat peer assumptions.
5. Keep group calls honestly unavailable.
