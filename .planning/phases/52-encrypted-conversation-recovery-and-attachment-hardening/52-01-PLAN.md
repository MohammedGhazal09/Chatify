# Phase 52 Plan 01: Recovery Key Helper Contract

## Objective

Add a strict, versioned local recovery-key contract for encrypted conversation secrets.

## Tasks

1. Add helper functions to `encryptedMessages.ts`:
   - export a recovery key for a chat that already has a local secret.
   - import a recovery key for a chat after validation.
   - return structured failure reasons for invalid imports.
2. Bind recovery keys to chat id and secret version.
3. Add Vitest coverage for:
   - export/import round trip.
   - missing secret export.
   - invalid prefix/version/chat id/base64/key length.
   - invalid imports preserving existing stored secrets.

## Verification

- `npm --prefix Frontend/Chatify run test -- encryptedMessages`
