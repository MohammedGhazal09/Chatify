# Plan 19-05: Product Polish Verification And Evidence

## Objective

Prove the notification, account/session, and state-polish work through focused tests, behavior-first Playwright checks, full frontend quality gates, and sanitized closeout evidence.

## Tasks

1. Read all Phase 19 summaries, `19-VALIDATION.md`, touched frontend tests, and existing Playwright smoke patterns.
2. Add a Phase 19 Playwright spec and fixture for notification settings states, muted background alert behavior, session-expired privacy, empty/no-results/offline states, and mobile layout.
3. Mock browser notification support/permission carefully in Playwright without depending on real OS notification UI.
4. Run focused tests from Plans 19-01 through 19-04.
5. Run full frontend tests, lint, and build.
6. Run behavior-first Playwright checks for Phase 19 and keep existing auth-expired privacy smoke passing.
7. Create `19-PRODUCT-POLISH-EVIDENCE.md` with command results, screenshots if generated, and residual release blockers.
8. Create `19-VERIFICATION.md` with each Phase 19 success criterion marked passed, blocked, or failed based only on evidence.
9. Update Phase 19 roadmap/status artifacts without claiming v1 release readiness.

## Verification

- `cd Frontend/Chatify; npm test -- --run`
- `cd Frontend/Chatify; npm run lint`
- `cd Frontend/Chatify; npm run build`
- `cd Frontend/Chatify; npm run test:ui -- --grep "Phase 19"`
- `cd Frontend/Chatify; npm run test:ui -- --grep "auth-expired smoke"`
- `npm run ops:check` if Phase 19 evidence or docs include operational blocker text.

## Success Criteria

Phase 19 closes with sanitized evidence proving the product-polish behavior locally, while release readiness remains blocked until the separate production smoke and call/TURN evidence exists.

## Must Haves

- No evidence artifact may include real credentials, smoke account passwords, tokens, or private message content.
- Playwright assertions for notifications must assert generic copy, not raw message previews.
- Final verification must preserve unresolved Phase 14, Phase 15, and Phase 17 release blockers.
