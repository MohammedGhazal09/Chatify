# Plan 22-01: Backend Group Chat Contract

## Objective

Add authenticated, CSRF-protected group creation by username with a hard 10-member cap.

## Tasks

1. Add group name/member constants and validation helpers.
2. Add `createGroupChat` controller for `chatName` and `memberUsernames`.
3. Resolve selected usernames with `validateUsername`, reject duplicates/self/invalid/missing/blocked users, and add the requester.
4. Create a group chat with `isGroupChat: true`, `groupAdmin: requesterId`, no `directKey`, and public member population.
5. Emit `chat:new` to all selected members and join all online member sockets to the new group room.
6. Add backend tests for success, min/max cap, duplicate/self/invalid/missing usernames, legacy email privacy, blocked member rejection, CSRF, and no email serialization.
