# Phase 41 Plan 01 - Locale Foundation And Direction Runtime

## Objective

Add the localization runtime, dictionaries, persisted preference, root language/direction updates, and date formatting helpers.

## Scope

- Add supported locale metadata and English/Arabic dictionaries.
- Add locale provider and translation/date helpers.
- Persist user locale preference in local storage.
- Wrap the frontend app with the locale provider.
- Add focused provider/runtime tests.

- [x] Date/time helpers format using the selected locale.
- [x] Focused locale runtime tests pass.
