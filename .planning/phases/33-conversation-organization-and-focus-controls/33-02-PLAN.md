# Phase 33 Plan 02 - Frontend Focus Controls And Continuity

## Objective

Add server-backed organization actions and sidebar filters without losing selected chat context.

## Scope

- Chat types and API client updates.
- React Query mutation for organization changes.
- Sidebar filter controls, filtered empty states, and stable pinned ordering.
- Conversation row indicators for pinned/starred/muted state.
- Replace local favorite storage with server-backed chat state.
- Component and hook tests for filtering and cache updates.

## Acceptance Criteria

- [ ] Users can archive, pin, star, and mute from conversation actions.
- [ ] Sidebar filters show all, unread, direct, group, archived, and starred views.
- [ ] Selected chat remains available when filter state would otherwise hide it.
- [ ] Chat cache updates after organization mutations and socket events.
- [ ] Frontend tests cover filter and action behavior.
