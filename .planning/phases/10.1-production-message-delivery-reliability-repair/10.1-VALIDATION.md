| Property | Value |

| **Quick frontend command** | `cd Frontend/Chatify; npm test -- --run src/api/messageApi.test.ts src/hooks/messageCache.test.ts src/hooks/useChatQueries.test.tsx src/hooks/useChatSocket.test.tsx src/pages/chat/components/MessageComposer.test.tsx` |

| **Local smoke command** | `cd Frontend/Chatify; npm run test:ui -- --grep "Phase 10.1 local delivery reliability"` |
| **Production smoke command** | `cd Frontend/Chatify; $env:CHATIFY_PRODUCTION_SMOKE='1'; npm run test:ui -- --grep "Phase 10.1 production delivery reliability"` |
| **Estimated runtime** | Targeted: 30-90 seconds per package. Full local: 3-8 minutes. Production smoke: environment-dependent. |

## Sampling Rate

- **After every backend task commit:** Run the quick backend command or a narrower touched-suite command.
- **After every frontend task commit:** Run the quick frontend command or a narrower touched-suite command.
- **After Wave 1:** Run full backend tests.
- **After Wave 2:** Run frontend tests, lint, and build.
- **After Wave 3:** Run local smoke and production smoke, or record a blocked production evidence result.
- **Before `$gsd-verify-work`:** Full backend tests, full frontend tests, lint, build, local smoke, and production evidence artifact must be present.
- **Max feedback latency:** 8 minutes for local validation; production smoke may exceed this only due to deployed service latency.
